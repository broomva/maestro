Pill-shaped action button; use for every clickable action — primary (ink fill) for the one main action, secondary/soft/ghost for the rest.

```jsx
<Button variant="primary">New mission</Button>
<Button variant="secondary">Connect</Button>
<Button variant="ghost" size="sm">Cancel</Button>
```

Variants: primary | secondary | soft | ghost. Sizes: sm (28) | md (36) | lg (44). Labels are sentence case, lead with the verb. Never Title Case, never emoji.
