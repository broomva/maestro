The Undertow — THE running signal. Wrap any matte card while its session runs; presence, not progress.

```jsx
<Undertow active={item.state === "running"}>
  <Card title="Weekly planning" meta="claude · 41m" />
</Undertow>
```

Requires styles.css on the page (the halo lives in tokens/motion.css). Never a colored border, never a spinner.
