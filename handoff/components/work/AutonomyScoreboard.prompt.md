The autonomy scoreboard — hours of unsupervised work with a notch per human look. The score that matters; never a completion percentage.

```jsx
<AutonomyScoreboard
  hours="6h 24m" sub="2 looks · longest run 3h 50m"
  segments={[{ start: 0, width: 34 }, { start: 36, width: 42 }, { start: 80, width: 14, live: true }]}
  notches={[34, 78]}
/>
```

Lives at the bottom of the sidebar in Maestro; works anywhere autonomy is the story.
