Segmented control — bordered pill of 2–4 mutually exclusive options; the Maestro settings pattern.

```jsx
<Segmented value={gate} onChange={setGate}
  options={[
    { value: "ask", label: "Ask first" },
    { value: "risk", label: "Ask on risk" },
    { value: "free", label: "Run free" },
  ]} />
```

Use Tabs for switching views, Select when options are many or long.
